## Starting 🚀

1. Clone or download the repository 

```bash
git clone https://github.com/emicasso/sCalculator
```
2. Change the working directory

```bash
cd sCalculator
```

3. Execute:

```bash
npm install
```

🌟 You have all you need! 🌟
